트위터

현재 사용 중인 설정은 저장되어 있지 않습니다. 설정 ID undefined의 설정입니다. undefined의 설정입니다.

설정을 바꾸고 저장해도 자동으로 백업되지 않고, 다른 설정으로 덮어쓰면 이전의 설정은 사라집니다.

설정 내보내기

트위터 계정을 사용하지 않거나, 설정을 공유하고 싶다면 익명 ID를 사용해서 설정을 저장할 수 있습니다.

설정 가져오기

저장된 설정으로 현재 설정을 덮어쓸 수 있습니다.

이유

OverlayPlugin이 오버레이가 다른 창을 띄우는 것을 허용하지 않아서, 직접 오버레이를 다시 실행해줘야 설정이 돌아오고 오버레이가 작동합니다.

설정을 변경하지 않았어도, 오버레이를 다시 사용하려면 아래의 단계를 따라야 합니다.

방법

  1. ACT 메인 창으로 갑니다.
  2. Plugins > OverlayPlugin.dll > kagerou
  3. 오른쪽 아래의 Reload overlay 버튼을 누릅니다.
  4. 설정 창이 자동으로 닫히고, 오버레이가 다시 나타납니다.
